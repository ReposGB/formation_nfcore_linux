Introduction

Ceci est le site compagnon pour la Formation en Bioinformatique Module 1 - Traiter ses données NGS (Single cell RNA-seq, RNA-seq bulk et DNA-seq) sous nf-core sur cluster de calcul.

Objectif : Savoir mener une analyse bioinformatique sur les séquences issues d’une analyse de séquençage à haut débit (NGS) dans un environnement Linux de type cluster en utilisant la bibliothèque de pipelines NF-Core.

A l’issue de la formation, les participants :

  • Seront familier avec la bibliothèque de pipelines NF-Core et le gestionnaire de pipelines Nextflow, ainsi que les principes de leur installation et configuration sur un cluster de calcul.
  • Sauront exécuter un pipeline standard NF-Core sur un environnement SLURM, en explorant les étapes et les résultats produits.
  • Pourront configurer et personnaliser des pipelines NF-Core à l’aide de paramètres avancés, en intégrant des solutions de conteneurisation (Docker/Singularity).
  • Sauront mettre en œuvre des cas pratiques d’analyse de données NGS avec NF-Core sur SLURM:
    • Identifier des variants génétiques (SNPs, indels) à partir de données DNA-seq avec le pipeline nf-core/sarek.
    • Quantifier l’expression génique à partir de données RNA-seq avec le pipeline nf-core/rnaseq.
    • Analyser l’expression génique à l’échelle de la cellule unique à partir de données scRNA-seq avec le pipeline nf-core/scrnaseq.
